# 💳 Predicting Credit Card Client Defaults (Taiwan)
This repository contains the end-to-end machine learning pipeline for predicting credit card defaults. Our goal is to build a classification engine capable of handling a highly imbalanced, 30,000-row financial dataset using rigorous data science and software engineering principles.

**Dataset Level:** Level 3 (+2 Bonus)
